Easystats

This is my most popular Gem to date. It extends the Ruby Array class and adds in neat features like weighted moving average and standard deviation.

Github - https://github.com/mgrigajtis/easystats
Ruby Gems - https://rubygems.org/gems/easystats

  • Easy to Install and use

    Many statistical methods available to use on arrays of numbers. Simply call the method, and read the output!

EasyZPL

I wrote this Gem as a wrapper for Zebra's ZPL\ZPL2 language. That way, you get Ruby's syntactic sugar instead of having to deal with tedious ZPL.

Github - https://github.com/mgrigajtis/easyzpl
Ruby Gems - https://rubygems.org/gems/easyzpl

  • Easy to Install and use

    Create Zebra labels using Ruby instead of ZPL.

jQuery Fireworks

A jQuery plugin that puts dazzling fireworks into a div.

Github - https://github.com/mgrigajtis/jquery-fireworks

  • Easy to use
    $('#divElement').fireworks();
    $('#divElement').fireworks({ sound: true, opacity: 0.4, width: '400', height: '300' });
